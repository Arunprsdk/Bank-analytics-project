
# Bank-analytics-project

## Project Overview
This Bank-analytics project analyzes transaction and client details data using SQL, Excel, Power BI, and Tableau to identify sales trends and business insights.

## Tools Used
- SQL
- Excel
- Power BI
- Tableau

## Project Workflow
1. Cleaned and transformed the data using SQL.
2. Performed analysis in Excel.
3. Built interactive dashboards in Power BI and Tableau.
4. Generated business insights and recommendations.
